# 🩸 Smart Blood Bank

A MERN Stack based Blood Bank Management System developed to manage blood donors, track blood inventory, and handle emergency blood requests.

##  Features

- Donor Registration
- Blood Inventory Tracking
- Blood Availability Search
- Emergency Blood Requests
- Real-Time Data Management

##  Tech Stack

### Frontend
- React.js
- Vite
- JavaScript
- HTML
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

##  Run Locally

```bash
git clone https://github.com/MadalaJeshnavi/smart-blood-bank.git

cd backend
npm install
npm start

cd ../frontend
npm install
npm run dev
```
